## 1. 语音功能

### Q1.1 语音读题（TTS）方案

A: 微信同声传译插件，已内置在微信小程序中。

### Q1.2 语音读题时是否同步显示字幕高亮？

A: 是，读题时同步显示字幕高亮。

### Q1.3 语音回答（ASR）交互逻辑

A: 用户说选项字母（"A"/"B"/"C"/"D"），系统识别后选中对应选项。

### Q1.4 语音与触控的关系

A: 语音答题和手动点击并存。

### Q1.5 是否支持方言/口音？

A: 不支持，只支持标准普通话，不标注。

### Q10 语音播报的触发方式

A: 语音读题有开关，默认自动播报，可关闭改为手动点击"听题"按钮。读题完毕后 ASR 自动监听，麦克风图标显示正在听，用户说完自动识别，无需按住按钮。

## 2. 后端 API

### Q2.1 API 完整清单

A: 只提供两个接口：
- 微信小程序登录
- 获取问题列表（分页，每页 100 条，按 `subject=1|4` 过滤）

答题结果、错题本等全部在前端本地处理。

### Q2.2 题目列表的拉取方式

A: 分页拉取，每页 100 条，支持按科目过滤（`subject=1` 或 `subject=4`）。

### Q2.3 答案数据的传输

A: 随题目一起下发。

### Q11 模拟考试模式下的答案保护

A: 前端自己隐藏答案字段，直到交卷才显示。

## 3. 数据同步

### Q3.1 同步方式

A: 全量覆盖。极速数据没有 ID，本地使用自增 ID；通过题目内容的哈希值判断是否需要更新。

### Q3.2 定时同步时间

A: 默认每天凌晨 3 点，可通过 `sync.cron` 配置。

### Q3.3 启动同步的幂等性

A: 启动时检查距上次同步时间，不足 24 小时则跳过。

### Q3.4 限频与失败处理

A: 超额后记录告警。失败自动重试，间隔和次数通过 `sync.retry_interval` / `sync.max_retries` 配置。

### Q14 极速数据 API 前置信息

A:
- 接口地址：`https://api.jisuapi.com/driverexam/query`
- 方法：GET / POST
- 请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `appkey` | string | 是 | API Key |
| `type` | string | 是 | 驾照类型（固定传 C1） |
| `subject` | string | 否 | 科目 1 或 4 |
| `pagesize` | string | 否 | 每页数量，最大 100 |
| `pagenum` | string | 否 | 页码 |
| `sort` | string | 否 | normal / rand |
| `chapter` | string | 否 | 章节（暂不使用） |

- 返回字段：`question`（题目）、`option1`–`option4`（选项）、`answer`（答案）、`explain`（解析）、`pic`（图片 URL）、`chapter`（章节）

## 4. 用户表

### Q4.1 用户表字段

A: 主键 `openid`，存昵称、头像，申请获取手机号。答题进度和错题本暂存前端本地。

### Q4.2 注册时机

A: 首次打开小程序时自动创建。

## 5. 答题交互

### Q5.1 练习模式 vs 模拟考试模式

A: 区分两种模式：
- **练习**：答完一题立刻显示对错和解析
- **模拟考试**：全部答完才出分，过程中不提示

### Q5.2 错题本机制

A: 答错自动收录，可按科目筛选，不区分分类。无复习提醒/推送。

### Q5.3 答题顺序

A: 默认顺序出题。用户可标记题目稍后回顾，标记后在错题本中有特殊标识。

### Q12 题目类型

A: 判断题、单选题。多选题已排除。判断题前端转为 A/B 单选题（A=对，B=错）。

### Q16 多选题的答案格式

A: 没有多选题。判断题转换为 A/B 单选题。

### Q17 分类（章节）列表的来源

A: 暂不使用 chapter 过滤。前端直接展示"全部"分类，后续需要再增加。

## 6. 微信登录

### Q6.1 Token 管理

A: JWT，存储在前端 storage。过期后尝试静默刷新，失败则需重新授权。

### Q6.2 多设备

A: 暂不支持，数据独立存储在本地。

## 7. 科目与分类

### Q7.1 分类体系

A: 科目 1 和科目 4 分类各自独立，数据从极速数据 API 动态拉取。

### Q7.2 跨科目查看

A: 不支持，需切换科目。

## 8. 配置文件

### Q8.1 配置项清单

A:

| 配置项 | 说明 | 必填 | 默认值 |
|---|---|---|---|
| `mysql.host` | MySQL 主机 | 是 | — |
| `mysql.port` | MySQL 端口 | 是 | — |
| `mysql.user` | MySQL 用户 | 是 | — |
| `mysql.password` | MySQL 密码 | 是 | — |
| `mysql.db_name` | 数据库名 | 是 | — |
| `jisuapi.key` | 极速数据 API Key | 是 | — |
| `wechat.appid` | 微信小程序 AppID | 是 | — |
| `wechat.secret` | 微信小程序 Secret | 是 | — |
| `sync.cron` | 定时同步 cron | 否 | `"0 3 * * *"` |
| `sync.on_startup` | 启动时是否同步 | 否 | `true` |
| `sync.retry_interval` | 重试间隔 | 否 | — |
| `sync.max_retries` | 最大重试次数 | 否 | — |

### Q8.2 示例配置文件

A: 需要提供 `config.example.yaml`。
